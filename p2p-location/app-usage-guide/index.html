App Usage Guide

This page gives a simple overview of how P2P Location works. It is intentionally brief so it stays accurate even as the app interface changes over time.

How it works

  1. One person creates a sharing session.
  2. The session owner shares the invite link or join code with another person.
  3. The other person opens the link in the app and joins the session.
  4. Both people allow the permissions needed for location sharing.
  5. Once connected, live location can be shared between the two participants.

Requirements

  • Both participants need the app installed.
  • Location services should be enabled on the device.
  • The app may require foreground, background, or notification permissions depending on the device and Android version.
  • An internet connection is needed to create or join a session.

Troubleshooting

  • If the invite link opens in the browser, try opening it again after the app is installed.
  • If location is not updating, check app permissions and device location settings.
  • If the session no longer works, it may have expired or been deleted.
  • If connection fails, retry from inside the app or create a new session.

Stopping sharing

You can stop participating in a session at any time by leaving it or deleting it, depending on your role in the session.
